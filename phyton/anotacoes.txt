PYTHON

 Tipos primitivos:
    int == número inteiro;
    float == número real;
    str == string ou escrita;
    boolean == true ou false.

 Variável:
    Exemplo: nome da variável recebe valor
     num = 1
    Podemos declarar qual o tipo primitivo, para dizer que será um número inteiro:     
     num = int(1)
    Para dizer que será um str:
     num = str(1)
      assim ficaria tipo
       num1 = str(1)
       num2 = str(2)
       total = num1 + num2
       12

     , end=''
